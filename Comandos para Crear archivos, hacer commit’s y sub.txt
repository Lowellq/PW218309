Comandos para Crear archivos, hacer commit’s y subirlos a GitLab

1. Crear carpeta en el escritorio donde estaba mi proyecto

2. cd (Ubicarme en la carpeta de escritorio donde creare mi proyecto)

3. Git init

4. Doten new mvc

5. Git status

6. Git add .

7. Git Status

8. git remote add origin git@gitlab.com:Angel.AMG/NombreDeProyecto.git (Es para agregar mi proyecto con GitLap)

9. Git status

10. Git commit -m ’Texto del commit’ (Es para crear commit’s)

NOTA: Puedo crear cosas en mi proyecto (Carpeta) y agregarlo con: git add (Nombre del archivo creado) y hacer su respectivo Commit (Esto es para tener mas comimos en mi proyecto).

Al final de hacer los commit, se deberá ejecutar el siguiente comando para subirlo al GitLab:

11. git push -u origin master (Es para subir el o los commit ya creados)
